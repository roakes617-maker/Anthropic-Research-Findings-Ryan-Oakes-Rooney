Three Goals I Set on Day One

When I inherited this account in April 2020, the relationship was at rock bottom. The previous team had destroyed trust with the CIO by running an "Illuminate" engagement as a vehicle to force a multi-million dollar expansion. I set three goals:

About Flex

Flex Ltd. is a $26B global manufacturing and supply chain company headquartered in Austin, Texas. With over 160,000 employees across 100+ facilities in 30+ countries, Flex is one of the world's largest diversified manufacturers — designing, building, and delivering products for some of the biggest brands in automotive, healthcare, industrial, cloud, and consumer electronics. Their scale and complexity made this one of the most strategically significant accounts in the territory.

02 — For Software Engineers

Claude Code vs. OpenAI Codex

Two different philosophies for AI-assisted development. Engineers are picking one or the other — here's what matters.

Dimension Claude Code OpenAI Codex
Philosophy Local-first, CLI-native. Runs in your terminal with full filesystem access. You watch it work and steer in real-time. Think "AI pair programmer sitting at your desk." Cloud-first, async delegation. Tasks run in isolated cloud sandboxes. Fire-and-forget, come back to a PR. Think "AI teammate working in parallel."
Primary Model Claude Opus 4.6 / Sonnet 4.5 — configurable per session GPT-5.3-Codex — optimized for agentic coding. 77.3 Terminal-Bench 2.0
Architecture CLI tool installed via npm. Runs locally on your machine. Direct filesystem + terminal access. No sandbox — full power, full responsibility. Cloud sandboxes preloaded with your repo. Internet disabled by default (optional live mode). Tasks return commits you review.
Parallelism Subagents + Agent Teams. Subagents (up to 7) handle focused tasks and report back. Agent Teams (experimental) are fully independent sessions that communicate directly with each other — can debate, challenge, and self-coordinate via shared task lists. Multi-thread cloud tasks. Multiple tasks run in parallel cloud sandboxes. Each task is isolated. No inter-task communication — coordination happens through you.
Context Continuity CLAUDE.md files — persistent per-project context files. Auto-loaded on every session. Claude remembers past decisions, architecture notes, team conventions across sessions. AGENTS.md files — similar repo-level config. Threads are resumable. Codex builds "memories" from past interactions for improvement over time.
Tool Integration MCP protocol (open standard) — 100s of integrations. GitHub, Slack, Jira, databases, Notion, Salesforce. Add servers via claude mcp add. Ecosystem growing rapidly. Skills system — bundled instructions + scripts for workflows. Figma design-to-code, Linear project mgmt, Cloudflare/Vercel deploy, image gen. Open-source skill library on GitHub.
Long Sessions Interactive pairing sessions. Context compaction keeps long conversations alive. Agent teams can run extended parallel explorations. Typical use: hours of active pairing. Tasks typically 1–30 min. GPT-5.3-Codex tested up to 7+ hours on complex tasks. Adapts thinking time dynamically based on complexity.
Surfaces Terminal (primary). VS Code, JetBrains IDEs. GitHub Copilot (as cloud agent via Copilot Pro+). Also available via API. ChatGPT sidebar (web), Codex CLI, VS Code IDE extension, Codex macOS app (Feb 2026 — command center for multi-agent orchestration), ChatGPT iOS.
CI/CD Via MCP integrations + GitHub Actions. Custom workflows through Claude Code SDK. Codex Autofix — automated CI failure investigation. Automations run on schedule (issue triage, release briefs, monitoring).
Security Model No sandbox — full local access. Permission modes (ask, auto-accept, yolo). CLAUDE.md can enforce per-project boundaries. Trust model mirrors giving a dev SSH access. Sandboxed by default. Internet disabled. Cached or live web search optional. Configurable per-project via settings. Open-source sandbox code.
Pricing Claude Max ($100–200/mo) for unlimited usage. Pay-as-you-go API billing also available. Teams/Enterprise included. Included with ChatGPT Plus ($20/mo), Pro ($200/mo for 6× limits). Business/Enterprise. API via Responses API.
Market Position ✓ 42% coding agent market share. $2.5B+ ARR. Used internally at Microsoft and Google. Fastest-growing developer tool in history. ~ 21% share. Strong async delegation story. Codex app positions it as "command center." Rapidly iterating (9 months of upgrades since launch).
